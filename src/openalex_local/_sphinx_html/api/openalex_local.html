openalex_local API

The openalex_local package exposes a small set of functions (search, get, count, info, configure, get_mode) and the Work / SearchResult dataclasses, plus the Config singleton from openalex_local._core.config. Each is documented below.

Core Functions

search

openalex_local.search(query, limit=20, offset=0)[source]

Full-text search across works.

Uses FTS5 index for fast searching across titles and abstracts.

Parameters:

  • query (str) – Search query (supports FTS5 syntax)

  • limit (int) – Maximum results to return

  • offset (int) – Skip first N results (for pagination)

Return type:

SearchResult

Returns:

SearchResult with matching works

Example

>>> from openalex_local import search
>>> results = search("machine learning")
>>> print(f"Found {results.total} matches")

Example

>>> work = get("W2741809807")
>>> work = get("10.1038/nature12373")
>>> print(work.title)
